# ApexPlanet Task 4 - Advanced Analytics & Statistical Modeling

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* SciPy
* Jupyter Notebook

## Tasks Performed

* Statistical Analysis (Mean, Median, Mode, Standard Deviation)
* Summary Statistics
* Hypothesis Testing (T-Test)
* Customer Segmentation using K-Means Clustering
* Predictive Modeling using Linear Regression

## Key Insights

* Sales and profit patterns were analyzed using descriptive statistics.
* Significant differences were identified between product categories.
* Customers were segmented into four groups based on purchasing behavior.
* A predictive model was developed to estimate sales from quantity data.
